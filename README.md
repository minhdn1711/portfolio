# Personalized Next.js Developer Portfolio

A more polished and recruiter friendly portfolio built with Next.js App Router and prepared for Vercel deployment.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Deploy to Vercel

1. Push this folder to GitHub
2. Import the repository into Vercel
3. Keep framework preset as Next.js
4. Deploy

## Important placeholders to replace

Edit `app/page.tsx` and update:

- `profile.name`
- `profile.email`
- `profile.github`
- `profile.linkedin`

## Notes

This version is personalized to the backend profile already described in the CV context:
- PHP Laravel backend development
- RESTful API work
- MySQL, SQL Server, Oracle
- Redis and RabbitMQ
- SMS and ZNS integration
- Linux VPS and Windows Server IIS deployment support

For full one to one accuracy, replace the placeholder company names and add exact employment dates from your final CV.
